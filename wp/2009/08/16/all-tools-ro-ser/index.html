全套仙境传说(RO)服务器工具下载 包含源代码

话说，已经有了服务端，要开服的话还必须会修改服务端才行，对照脚本，txt一个一个的改实在不便，所以有了达人制作了配套服务端修改的工具，让我们可以自己制作自己喜欢的仙境传说。

难以置信，离开RO已经多年，至今还不断有人找我要当年我写的这些软件（当时网名开裆酷）～

这些软件随着我的某次格式化消失了，我 自己就没有再去留意它们，不过今天有人找我索要时提醒说G宝盘里有，但是他下不来，我这才想起还有这个么个网站，但是里面能找到程序也还是非常有限，其中 Athena Mission Center还是当时经过加密的，现在已经不能再使用了，其他软件也没有源代码，不能保证还能继续适用。

到这里我突然想到可能以前曾经刻录留存过一部分软件，于是开始翻那堆刻录盘，经过一下午，总算是找得比较全乎了，在此发布出来，供冲着RO而来的朋友下载。

(里面的东西很乱，有源代码、测试内容等等，自己找找能用的吧，都不能用就尝试改源代码重新编译)

Athena Script Helper

脚本编写助手，应该算是本人的成名作吧，呵呵。

http://www.uushare.com/user/icesee/file/1824491

Athena Data Center

数据管理软件，与上面那个齐名，都有极高普及率。

http://www.uushare.com/user/icesee/file/1824476

Athena Offline Announcer

用于不登录游戏客户端而直接发布公告、悄悄话。

http://www.uushare.com/user/icesee/file/1824536

Athena Mission Center

充分参考魔兽世界任务系统后创造的任务脚本生成系统，以统一的架构实现了极多的功能扩展，不写一句代码就可以实现复杂的任务创建，大幅超越官方那套简单而乏味的任务机制。

http://www.uushare.com/user/icesee/file/1824480

Athena Data Box

这是未完成的作品，目的是让文本数据向数据库转换，继而可以通过SQL语句查询、更新数据，并将更改后的数据重新转换回文本。

http://www.uushare.com/user/icesee/file/1824473

Athena Run Server

服务器启动器。

http://www.uushare.com/user/icesee/file/1824481

Athena Seller

未公开的软件，用于生成以特定的道具换购物品或其他效果（如经验值、等级、外观）的脚本，一般是GM向玩家卖这个特定道具，然后玩家用此道具换自己想要的装备。

http://www.uushare.com/user/icesee/file/1824493

Athena Battle Modifier

用于修改服务器全局参数设置。

http://www.uushare.com/user/icesee/file/1824470

SearchRex

正则表达式替换工具，现在已升级为Regex。

RegeX是一款正则表达式辅助程序，它通过高亮匹配及替换内容使整个流程可视化，帮助用户更好地理解和使用正则表达式这一强力工具。

RegeX定位为免费软件，任何人都可无限制的使用它。

它的前身叫做SearchRex，此前我曾在一个小圈子里推广过它，很受好评。

我自己就是它最最忠实的铁杆用户：它帮我逐步学习和掌握了正则表达式；它帮我完成了很多文字处理工作；每次关于正则表达式编程的测试工作也都是依靠它来完成的。

经过一段时间的WPF技术学习后，我决定用WPF技术对它进行一次升级，于是就有了RegeX：
